# Customer Demographic Segmentation Analysis

## 📌 Project Overview
This project features a demographic profiling model designed to process and analyse over 1,000 consumer records. The objective was to uncover the core socio-economic drivers behind purchasing behaviour to help marketing and product teams optimise their customer acquisition costs (CAC).

## 🛠️ Tools & Technologies Used
* **Advanced Excel:** Data transformation workflows and multi-dimensional pivot models.
* **Data Visualisation:** Custom Bar Charts and Column Charts for demographic comparison.
* **Conditional Logic:** Built custom IF/AND/OR workflows for lifecycle bracketing (age, income groups).

## 🎯 Key Segmentation Insights
* **The High-Value Segment:** Discovered that **middle-aged individuals (ages 31–54)** represent the primary revenue engine, purchasing bikes more frequently and in significantly higher volumes than any other demographic.
* **The Low-Performing Segment:** In contrast, the data revealed that consumers **under the age of 30** purchase very few bikes, signaling a low-converting group.
* **Business Value:** Provided data-backed target profiles to marketing teams, allowing them to shift ad spend toward the high-converting 31–54 age bracket and lower overall acquisition costs.

## 📂 Repository Structure
